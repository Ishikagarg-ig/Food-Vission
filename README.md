# Food-Vision
App aims to stop <b>food wastage</b> with the help of <b>woman</b> for the prevention of <b>irregular climate changes</b>.

Won <b>2nd</b> prize in <b>international hackathon</b>.

Developed for Android using <b>Java</b>.
